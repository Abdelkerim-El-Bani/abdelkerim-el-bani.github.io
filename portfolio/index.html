Design of an Instrumented Glove for Myoelectric Hand Prosthesis

An instrumented glove system for real-time data acquisition and control of a myoelectric hand prosthesis using sEMG sensors and machine learning.

sEMG Sensors Machine Learning CNN
Click to expand

Developed an innovative instrumented glove (Vendetta) designed to provide intuitive control of a myoelectric hand prosthesis. The system captures hand movements and muscular activity through flex sensors, an IMU, and sEMG electrodes. Data is collected and processed using the Ninapro database, comprising 10 able-bodied subjects and 2 amputees for real-world validation. The project implements a two-stage machine learning approach: gesture classification using a 2D CNN model achieving 83.98% accuracy for 10 movement classes, and angle prediction using specialized regression models with an average MAE of 7.54°. The system successfully demonstrates real-time prosthesis control based on movement recognition and intent prediction.

Implementation of FOC for a Brushless Motor using STM32G4

Field-Oriented Control implementation for brushless DC motors using STM32G4 microcontroller with FreeRTOS and custom Qt SDK.

FOC STM32G4 FreeRTOS
Click to expand

Completed an internship project focused on implementing Field-Oriented Control (FOC) for brushless DC motors using the STM32G4 development board. The project involved theoretical research on FOC principles, practical setup with the P-NUCLEO-IHM03 motor control pack, and development of a simplified FOC framework optimized for embedded systems. Key achievements include FreeRTOS task integration for reliable multitasking, ASPEP debugging protocol implementation for real-time motor communication monitoring, and a custom Qt SDK application for intuitive motor control and visualization. The system was profiled and optimized using advanced debugging tools, resulting in a functional, adaptable platform for motor control applications.

High-Speed Competition Line Follower Robot

A high-performance line follower robot with 8 IR sensors, ESP32, and multiple control algorithms including PID and path learning.

ESP32 PID Control IR Sensors
Click to expand

Designed and built a high-speed competition line follower robot featuring 8 infrared sensors and an ESP32 microcontroller. The robot implements three distinct control variations: a PID-based algorithm for real-time line tracking, a path learning mode that records and optimizes trajectories, and a manual control interface for testing and calibration. As the project lead, I supervised the construction of 10 units within the electrical engineering club, overseeing the entire assembly and integration process. The chassis was custom-designed with plexi components that were precision-cut using a CNC laser, ensuring consistency and quality across all units. The project demonstrates expertise in sensor integration, control algorithms, and team leadership in a competitive robotics environment.

Manette - Multi-Protocol Wireless Controller

A versatile wireless controller with dual joysticks and buttons supporting multiple communication protocols.

ESP-NOW WiFi Bluetooth
Click to expand

Originally designed as a drone controller with dual joysticks and multiple buttons, Manette evolved into a comprehensive testing platform for wireless communication protocols. The device supports three distinct communication modes selectable via button press: Mode 1 implements ESP-NOW for low-latency peer-to-peer communication, Mode 2 utilizes WiFi for extended range connectivity, and Mode 3 employs Bluetooth for broad device compatibility. Each protocol is fully integrated and can be switched seamlessly during operation, making it an ideal platform for protocol evaluation, testing, and comparative analysis. The project demonstrates proficiency in multiple wireless technologies and the ability to design flexible, multi-protocol hardware interfaces.

Embedded Multitasking Platform (FreeRTOS, MQTT)

A comprehensive real-time embedded system on STM32 with FreeRTOS, MQTT integration, and SCADA capabilities.

FreeRTOS STM32 MQTT
Click to expand

Architected and implemented a complete real-time system on STM32 with FreeRTOS, integrating advanced multitasking management. The system runs entirely on an STM32 microcontroller with an ESP NodeMCU paired for WiFi connectivity. Despite being in active development, the platform already supports USB keyboard input, multiple applications including email and sensor monitoring, and features a vibrant OLED display with colorful LED indicators. Built on FreeRTOS, the system provides a compact yet powerful playground for real-time multitasking, embedded systems development, and IoT integration. The architecture is designed for extensibility, with plans to incorporate additional applications, advanced sensors, and enhanced connectivity options. This project showcases expertise in real-time operating systems, embedded systems design, and the ability to create scalable, feature-rich platforms on resource-constrained devices.
